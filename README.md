# Project Name
mygre-rust

## Description

This is an cli utility similar to grep to practice rust language skills

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Installation

Download repo and run `cargo build`

## Usage

mygrep-rust <options>

- --pattern -w
- --path -p
- --replace -r
- --new_file -n
